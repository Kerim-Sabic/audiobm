# Local SEO Operations Checklist — Svijet Sluha

Manual tasks for the business team. These are not code tasks and must use only real branch information, real photos, and genuine customer feedback.

## Google Business Profile

- Claim or create one profile per real branch: Banja Luka, Gradiška, Bijeljina, Doboj, Brčko, Sarajevo, Tuzla.
- Use the real signage/legal branch name, likely Audio BM [Grad], unless signage changes.
- Use exact NAP from the live branch page and CMS. Do not publish Sarajevo until address/phone are confirmed.
- Add the branch URL with UTM: /poslovnice/[slug]?utm_source=google&utm_medium=gbp.
- Add appointment URL: /zakazivanje?utm_source=google&utm_medium=gbp.
- Confirm categories: Hearing aid store as primary; Audiologist and Medical supply store as secondary where available.
- Upload real photos for each branch: exterior/signage, entrance, reception, test room, team, products, accessibility/parking if relevant.
- Add services only if actually offered at that branch.
- Publish weekly GBP posts: free hearing check, educational tips, branch news, service reminders.

## Bing Places and Maps

- Import verified Google profiles into Bing Places where possible.
- Check Apple Maps, HERE, Waze, and major local directories for consistent NAP.
- Keep the same spelling, phone formatting, and URLs everywhere.

## Reviews

- Ask only genuine customers for reviews after real visits.
- Use QR cards/SMS links per branch; never incentivize fake or scripted reviews.
- Reply to every review professionally within 3 business days.
- Escalate medical/service complaints to the responsible branch before replying publicly.
- Track review count, average rating, reply rate, and recurring service issues monthly.

## Citations and Local Authority

- Build citations on BiH business directories, health portals, local chamber pages, and municipality/business listings.
- Seek local backlinks through verified partnerships: senior associations, hearing-health education events, clinics, pharmacies, schools, music/industry safety partners.
- Keep press releases factual and link to the relevant branch/service page.

## Search Console / Webmaster Tools

- Use Google Search Console Domain property for svijetsluha.com.
- Submit sitemap.xml as a relative sitemap path inside the property.
- Set up Bing Webmaster Tools and submit sitemap.
- Monitor indexing for /slusni-aparati, /poslovnice, all branch pages, /zakazivanje, /online-test-sluha, and key services.

## Monthly Reporting

- Export Search Console queries/pages and compare impressions, clicks, CTR, and average position.
- Review GBP calls, website clicks, direction requests, photo views, and reviews per branch.
- Check sitemap errors, indexed/not-indexed reasons, and 404s.
- Update the content plan based on real user queries, not guessed keyword stuffing.
